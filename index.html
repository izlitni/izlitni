About

Operating under the pseudonym "izlitni", I build applications that solve real problems — from fintech platforms to cryptographic toolkits, from prayer time apps to social experiments. No face, no brand, just consistently shipped work.

My focus spans full-stack development, security tooling, and minimalist interface design. I believe the best software doesn't need a face behind it — it just needs to work. Every project is a node in a larger network, each one self-contained but connected by the same philosophy.
